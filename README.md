# React Project with Vite

## Langkah-langkah Instalasi

1. **Clone Repository:**

   ```bash
   git clone https://github.com/farendivaz/react-frontend.git
   cd react-frontend
   ```

2. **Instalasi Dependensi:**

   ```bash
   npm install
   ```

3. **Menjalankan Server**

   ```bash
   npm run dev
   ```

   Server akan berjalan di `http://localhost:5173`.
